Chrome's Autoplay Policy

Elemental 4 plays music and sounds. These are enabled by default when playing the game. On chrome, to play audio you must first have the user interact with the page, with a click or tap on the page. This is fine since theres a a "Play Game" button on the homepage. However, once you play the game once, the homepage will automatically start the game, meaning the audio will fail to play, as there has been no user interaction in that session.

Autoplay can be enabled if Chrome thinks the user frequently uses media on the page. To make this easier, all pages on Elemental4.net embed an 8 second 30kb audio file that is almost impossible to hear. This makes sure almost every session is counted as one that the user used media on, leading the game to be user whitelisted within about 6 sessions.

You cannot disable this autoplay bypass measure, but you can disable game music and startup sound in the game settings if you do not like it.
