Combine two elements to make a new element, if a combination doesn't currently create any element, you can suggest what element it should create. It is exactly how Elemental 3 worked, but it is not running anymore, so I created my own version.

This is a revival of an old project, supercharging it from a medium product to something amazing.

The "Old Elemental 4" was a re-creation of the Elemental 3 game. At that time, I was a bit newer to web development, so the client program used the material.io user interface library. It's a great library, but for a game like this, the UI should be completely custom. Once I discontinued the database due to hosting restrictions, and the growing size of the database, I made a client-only version of the game based on modding, Elemental Lite.

This "New Elemental 4" project was supposed to be adding the Elemental Lite client to the already existing Elemental 4 Server, but it soon became more of a full rewrite of the game, with Elemental Lite's basics as a starting point. Full animations, Themes, Statistics, Custom Servers, and Keyboard shortcuts were built from the ground up for this project, making it take weeks longer than expected. The final product is awesome though.
